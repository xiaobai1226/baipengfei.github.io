    Linux配置本地yum源

    高压锅里的小白 2021/8/17 Linux

      工作中会使用到内网机器，无法连接互联网，这样就不能方便的配置线上的yum源安装软件，故整理出这篇教程。

      其实原理是把yum源设置成iso文件，然后通过yum来进行安装，所以此方法仅能安装iso镜像中包含的软件，若想安装其他iso不包含的软件，比如docker等还需自行下载相关文件进行安装。

    # 前期准备

    1. 一台yum系的linux服务器
    2. 与linux主机系统对应的iso系统镜像文件（版本最好比系统版本高或一致）

    # 操作步骤

    1. 将iso镜像文件上传至linux服务器
      上传后，iso镜像文件所在位置为~/rhel-server-7.8-x86_64-dvd.iso

    2. 执行命令将iso镜像挂载到/mnt/cdrom目录下
      若/mnt目录下没有cdrom文件夹，则需先手动创建一个目录。

      mount -o loop 镜像文件 /mnt/cdrom
本文为 mount -o loop ~/rhel-server-7.8-x86_64-dvd.iso /mnt/cdrom
      1
      2

      挂载成功之后 可以df -h看到 iso 文件成功挂载到指定的目录下。

      若想要解除挂载，可使用如下命令（本篇教程无需执行）

      umount /mnt/cdrom
      1

    3. 设置yum源为本地iso文件
      修改/etc/yum.repos.d/rhel-source.repo文件进行yum源设置
      注：/etc/yum.repos.d/路径下，后缀为repo的文件均可

      [rhel-source]
name=Local Source
baseurl=file:///mnt/cdrom
enabled=1
gpgcheck=0
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-redhat-beta,file:///etc/pki/rpm-gpg/RPM-GPG-KEY-redhat-release
      1
      2
      3
      4
      5
      6

    4. 更新yum

      yum update
      1

    至此，就可以用yum install xxx来安装之前提示未安装的那些依赖程序了。
